مجموعه ویجت های حرفه ای برای Elementor - شامل نقشه ایران، جدول المنتور و اسلایدر تصویر

== Description ==

افزونه WPVision Addons مجموعه‌ای از ویجت‌های حرفه‌ای و کاربردی برای Elementor است که شامل موارد زیر می‌شود:

* **نقشه ایران تعاملی**: نقشه SVG تعاملی با قابلیت سفارشی‌سازی کامل
* **جدول المنتور**: جدول حرفه‌ای با طراحی زیبا و مدرن
* **اسلایدر تصویر**: اسلایدر پیشرفته با انیمیشن‌های چرخشی

== Installation ==

1. فایل zip افزونه را در پنل مدیریت وردپرس آپلود کنید
2. افزونه را فعال کنید
3. از منوی WPVision Addons تنظیمات را مدیریت کنید
4. ویجت‌ها را در Elementor استفاده کنید
